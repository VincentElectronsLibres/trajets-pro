Description

Application de suivi kilometrique professionnel pour les membres de l ASBL Electrons Libres et le personnel de la Commune de Nandrin.

Mode d emploi

  • + Trajet : touche un bouton pour enregistrer. Utilise les fleches pour changer de mois.
  • Historique : consulte tes trajets et genere le releve XLS ou envoie par e-mail.
  • Parametres : gere tes trajets, l index et ton compte.

Confidentialite

Toutes tes donnees sont stockees uniquement sur ton appareil. Aucune donnee n est envoyee a un serveur tiers, sauf les e-mails via EmailJS.
