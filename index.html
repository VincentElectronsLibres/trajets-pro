📋 Description

Application de suivi kilométrique professionnel pour les membres de l'ASBL Electrons Libres et le personnel de la Commune de Nandrin. Permet d'encoder rapidement ses trajets et de générer les notes de remboursement mensuelles.

📖 Mode d'emploi

  • + Trajet : touche un bouton pour enregistrer un trajet. Navigue avec ‹ › pour changer de mois.
  • Historique : consulte tes trajets, le total km et le remboursement. Génère le XLS ou envoie par e-mail.
  • Paramètres : gère tes trajets, l'index kilométrique et ton compte.

🔒 Confidentialité

Toutes tes données sont stockées uniquement sur ton appareil. Aucune donnée n'est envoyée à un serveur tiers, sauf les e-mails via EmailJS.
